Why do APIs return JSON instead of plain text?
JSON is a structured data format that is easy for different applications and programming languages to parse and process. It allows us to send multiple pieces of information (like status, message, and data) in a single response.

----------------------------
Request Body
Request Body contains data sent from the client to the server.
It is mainly used with POST and PUT requests.
Most modern APIs send data in JSON format.
The server reads this data using req.body.

express.json()
Built-in Express middleware.
Converts incoming JSON into a JavaScript object.
Makes data available inside req.body.
Must be written before routes.
