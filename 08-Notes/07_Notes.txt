JWT= Jason Web token
JWT use karte hain taaki user ko har request mein email aur password na bhejna pade. Login ke baad server ek token generate karta hai, aur future requests mein user us token ko bhejta hai. Server us token ko verify karke user ki identity confirm kar leta hai.
JWT authentication stateless hoti hai. Token client side store hota hai, aur har request ke saath client us token ko server ko bhejta hai. Server token ko verify karta hai, store nahi karta.
